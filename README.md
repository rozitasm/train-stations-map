# Train Stations Map 

The app displays train stations in Germany on an interactive map using React and Leaflet
---

## Features

- Fetches station data from an API
- Displays stations on a map
- Filters stations by city
- Clicking a station zooms the map to its location
- Loading and error states
- Basic test included

---

## Tech Stack

- React
- Leaflet.js / React Leaflet
- JavaScript
- Tailwind CSS
- Jest & React Testing Library
